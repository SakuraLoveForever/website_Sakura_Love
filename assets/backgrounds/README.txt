Put your background images in character folders.

assets/backgrounds/violet/
assets/backgrounds/toki/

Inside each folder, name images like:
1.jpg
2.jpg
3.jpg
...

The website randomly uses one image from the selected character folder.
Make sure to update the image count in script.js -> characterImageCount.

Tips:
- Use JPG format (.jpg) for all files.
- Keep image size moderate (for example around 1920x1080) for faster loading.
- After adding files, refresh the page to see changes.
